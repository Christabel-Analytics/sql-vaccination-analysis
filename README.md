# Vaccination Data Analysis using SQL

## Project Overview
This project analyzes childhood immunization data using PostgreSQL to evaluate vaccination coverage, vaccine distribution, facility performance, and immunization trends. The goal is to generate insights that can support evidence-based public health planning and improve immunization programs.

## Objectives
* Analyze childhood vaccination records.
* Evaluate immunization trends over time.
* Assess vaccine distribution across different vaccine types.
* Compare vaccination performance across states and health facilities.
* Identify patterns in vaccination timeliness and completion.

Key Questions Answered

1. How many children were vaccinated?
2. How many vaccine doses were administered?
3. What is the distribution of vaccine doses by vaccine type?
4. What are the yearly vaccination trends?
5. How has the administration of each vaccine changed over the years?
6. Which states recorded the highest vaccination coverage?
7. Which health facilities were the top-performing vaccination centers?
8. What proportion of vaccinations were administered on time versus delayed?
9. What is the vaccination completion status of children?
10. Which state administered the highest number of vaccines?
11. What is the average number of vaccine doses administered per child?
12. what is the Top vaccine adminstered?

Key Insights

* Identified overall childhood vaccination coverage.
* Evaluated annual immunization trends.
* Compared vaccine uptake across different vaccine types.
* Ranked states and health facilities based on vaccination performance.
* Assessed vaccination timeliness and completion rates.
